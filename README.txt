Projeto para identificação de fraude em dados financeiros e de e-mails (case da empresa de energia Enron, 2001) através de estudo comparativo entre três diferentes algoritmos de machine learning (Decision Tree, SVC e AdaBoost) passando pelas etapas de treinamento e validação por meio das métricas (recall, precision e F1). Bibliotecas utilizadas: scikit-learn, numpy e matplotlib.

Versão
- python 2.7.16
- ipython 5.8.0
- numpy 1.11.3
- scikit-learn 0.19.2
- matplotlib 1.5.1

Fazem parte dessa pasta:
- poi_id.py: código inicial do identificar de pessoas de interesse (POI, Person of Interest)
- my_functions.py: código com as funções calc_metrics(), computeFraction(), feature_generate(), ranking_feature()
- final_project_dataset.pkl: conjunto de dados para o projeto
- tester.py: código de teste
- my_dataset.pkl
- my_classifier.pkl
- my_feature_list.pkl
- questionario.pdf
